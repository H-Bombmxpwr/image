What this is

A serious browser editor without the usual cloud-only drag.

Image Lab is built to keep common editing work fast in the browser while still offering heavyweight server tools like seam carving and AI background removal. The goal is simple: make the app feel powerful without making every click wait on the network.

Local-first speed

Crop, resize, rotate, filters, adjustments, and many exports now stay in the browser.

Heavy tools when needed

Seam carving and rembg-powered background removal stay server-backed for better results.

Metadata-aware workflow

Edit metadata in the UI and embed it into the file when you export.

Real GIF support

Resize, trim, reverse, ping-pong, optimize, and export individual poster frames.

Stack

Flask, Pillow, NumPy, Cropper.js, seam-carving, and rembg.

Design goal

Make a browser editor that feels fast enough to keep open all day, but capable enough to handle difficult image jobs too.
